Quick Reference

Color Usage

  • ✓ Primary (Teal): Main actions, brand
  • ✓ Secondary (Blue): Alternative actions
  • ✓ Green: Success, approvals
  • ✓ Red: Danger, deletions
  • ✓ Yellow: Warnings, pending

Button Hierarchy

  • 1. Primary Button: Most important action
  • 2. Secondary Button: Less important
  • 3. Outline Button: Tertiary actions
  • 4. Ghost Button: Minimal actions
