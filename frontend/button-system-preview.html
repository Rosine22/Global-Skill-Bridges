Quick Decision Guide

🎯 Use PRIMARY (Teal) for:

  • ✓ Submit forms
  • ✓ Save changes
  • ✓ Create new items
  • ✓ Post jobs
  • ✓ Send messages
  • ✓ Get Started buttons

ℹ️ Use SECONDARY (Blue) for:

  • ✓ View Details
  • ✓ Learn More
  • ✓ Back/Cancel (non-destructive)
  • ✓ Info buttons
  • ✓ Filter/Search

✅ Use SUCCESS (Green) for:

  • ✓ Approve applications
  • ✓ Accept requests
  • ✓ Confirm actions
  • ✓ Complete tasks
  • ✓ Verify information

⚠️ Use DANGER (Red) for:

  • ✓ Delete items
  • ✓ Reject applications
  • ✓ Remove users
  • ✓ Cancel (destructive)
  • ✓ Decline requests
